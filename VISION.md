Маленький проект онлайн маркетплейс
 Видение проекта:

 Интернет-магазин должен иметь меню:

  1. Главная страница:
    что из себя представляет страница: 
        показать заглавную страницу отсюда весь функционал магазина строится это точка принятия
        действий от пользователя и дальнейшее передвижение по магазину:
      функционал показа топ товаров в магазине чтобы страница не была пустая с одними
        командами передвижения по магазину 

        1. Будет составлять топ 5 товаров по продаваемости на сайте
        2. Функционал добавления в корзину из топ 5 товаров из главной страницы
  3. Показ самих страниц для передвижения нумерацией 
        4. Функционал перехода на другие страницы
  5. Функционал выхода из магазина

 2. Страница категорий:
  что из себя представляет страница: 
  Показать все категории товаров в магазине показать 
  товары по категориям по умолчанию: smartphones
  изменять по желанию пользователя категорию и выдавать результат смены категории 
  
       1. Показ категорий перечисление из ассортимента для выбора:
          1. smartphones
          2. laptops
          3. computers
        2. Показ категории для выбора по умолчанию: smartphones

             1. показ самих элементов smartphones для показа
             2. Функционал добавления самих вещей в корзину
             3. Функционал для принятия изменения других категорий по выбору пользователя
             4. Функционал для перехода на другую страницу
             5. Функционал выхода из магазина

  3. Страница поиска:
  что из себя представляет страница: 
       Показ всех товаров по умолчанию так как не совершен поиск еще
       если поиск совершен то показа найденных товаров если поиск не удачный то выдать
       поисковый запрос не найден

       1. функционал показ всех товаров по умолчанию
       2. Функционал для перехода на другую страницу
       3. Функционал добавления в корзину товаров
       4. функционал совершения поиска
       5. функционал удаления поискового запроса
       6. функционал выхода из магазина

4. Страница корзины
  что из себя представляет страница:
      Показать страницу корзины если она пустая предложить добавить в корзину товаров
      если товар в корзину добавлен то показать сумму цен корзине по добавленным товарам
    дальше ждать действий пользователя для удаления товара из корзины или оформление товара из корзины
      1.Показ самой корзины что там есть 
      2. функционал перехода на другие страницы 
      3. функционал оформления заказа
      4. функционал удаления товара из корзины
      5. функционал выхода из магазина
